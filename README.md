# n8n LinkedIn Content Automation

This repository contains an n8n workflow that automates LinkedIn content handling using AI.

## What it does
- Checks a Notion database for scheduled LinkedIn posts
- Uses an AI agent to process content
- Sends email notifications

## Tools Used
- n8n
- Notion
- AI (LLM)
- Gmail

## Setup
Credentials are not included.
To use this workflow, connect your own:
- Notion
- Email provider
- AI model
